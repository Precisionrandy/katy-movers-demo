What just happened when you submitted that quote

  1. Your form data was sent to a real backend API (/api/quote) — the same shape of payload a production system would send from a live AI phone call.
  2. The server scored the lead using the same logic that would run after a real VAPI call: move size, distance, special items, and timing all factor into the estimate.
  3. A structured record was created — in production this becomes a contact card in the Precision Partners dashboard, exactly like the ones you've seen for Matthew's HVAC Co.
  4. In the real version, this same flow happens automatically on every phone call: Twilio receives the call, VAPI runs the conversation, N8N orchestrates the follow-up, and Supabase stores everything.
